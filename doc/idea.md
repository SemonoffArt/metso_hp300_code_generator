## Идея приложения генерации кодов для инженерного меню METSO HP300

### Цель
Создать простое графическое приложение, которое позволяет оператору/инженеру получать код доступа в инженерное меню дробилки METSO HP300 на основе кода, отображаемого на операторской панели.

### Краткое описание
- Пользователь запускает приложение на ПК.
- На экране отображается поле ввода для кода, выданного операторской панелью дробилки.
- Пользователь вводит этот код и нажимает кнопку (например, «Сгенерировать» / "Generate").
- Приложение вычисляет соответствующий код доступа в инженерное меню.
- Приложение показывает рассчитанный код, который можно ввести на дробилке.

### Основные функции
- **Ввод исходного кода**: текстовое поле для ввода кода, отображаемого на панели METSO HP300.
- **Вычисление кода доступа**: реализация алгоритма преобразования исходного кода в инженерный (будет уточнён и реализован отдельно).
- **Отображение результата**: вывод рассчитанного кода в удобном виде (крупный шрифт, возможность копировать/переписать).

### Ограничения и предположения
- Алгоритм генерации кода доступа известен или будет получен/восстановлен отдельно и встроен в приложение.
- Приложение работает локально (без обязательного доступа к интернету).
- Интерфейс ориентирован на быструю и удобную работу оператора.

### Возможные направления развития
- Добавить журнал (лог) с историей введённых кодов и рассчитанных результатов.
- Реализовать поддержку разных моделей дробилок, если алгоритмы генерации кодов отличаются.
- Добавить настройки языка интерфейса (например, русский/английский).
